How to Use Templates

2 min read · Jul 2026

Templates are the fastest way to get a good visual in Grok Imagine. You do not need to write a prompt from zero.

Steps

  1. Open Templates.
  2. Filter Free / Premium if you want, then choose a category.
  3. Click a template card → opens in Grok Imagine.
  4. Add your product, photo, or idea.
  5. Generate 3 to 5 versions (hit Redo for more variations).

Which category to pick

  • Product - ads, e-commerce, launch images.
  • Mockups - packaging, apparel, posters, brand scenes.
  • Style Edit - cinematic, fantasy, vintage, dramatic edits (all free-labeled).
  • Filters - color grades and mood changes.
  • Make-up - beauty looks.
  • Common Uses - cleanup, portraits, pro shots.

Free vs Premium + Super Grok

Free / Premium badges describe the look tier in Imagine - not an X Freeze checkout lock. To run templates you need Super Grok or Super Grok Heavy on Grok’s side. Browsing X Freeze is free.

Best tip

Do not stop at one result. Generate a few versions, keep the best still, then animate it with a motion prompt.

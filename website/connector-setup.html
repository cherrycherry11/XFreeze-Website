Guide

Connector setup

Connector skills talk to real apps - Gmail, Outlook, Google Calendar, Canva, and more. Instant skills do not need this step.

Grok

  1. Type /mcps in chat.
  2. Find the app you need (Gmail, Google Calendar, Microsoft Outlook, Canva, etc.).
  3. Turn it on and complete sign-in when prompted.
  4. Press r in /mcps to refresh the tool list.
  5. Run a quick test - e.g. ask Grok to list today's calendar events or summarize unread mail.

Advanced: add servers in ~/.grok/config.toml under [mcp_servers], then refresh /mcps.

Cursor

  1. Open Cursor Settings → MCP (or edit your MCP config file).
  2. Add the server for the app you need - Gmail, GitHub, Slack, etc.
  3. Authenticate when Cursor prompts you (browser OAuth or API token).
  4. Restart or reload MCP so tools appear in the agent.
  5. Paste the skill install prompt; the agent can then call live tools.

Each MCP server has its own install docs. Use the official connector for the app when available.

Claude (Desktop)

  1. Open Settings → Integrations / Connectors.
  2. Enable the connector for the service you need.
  3. Sign in and confirm permissions.
  4. Paste the skill install prompt from X Freeze, then use the slash command.

Troubleshooting

Skill installed but nothing happens
Connector is probably off or not signed in. Open /mcps (Grok) or MCP settings (Cursor) and verify green/authenticated status.
AI says it cannot access my mail
That is expected without a connector. The skill is working - the integration is not connected yet.
Do I need connectors for every skill?
No. Use the Instant only filter on the skills page for skills that work from text alone.
Is my data sent somewhere else?
Connector skills use your AI tool's official integration. X Freeze only gives you the SKILL.md instructions - I never see your mail or files.

How to use (copy & paste)

Instant skills no setup

  1. 1Set Type → Instant (and Free if you want) or open a pack tagged Instant.
  2. 2Copy prompt - includes the full SKILL.md file.
  3. 3Paste into Grok, Cursor, or Claude. It creates the skill file for you.
  4. 4Run the slash command (e.g. /meeting-notes) or describe the task.

Connector skills needs apps

  1. 1Read the connector setup guide for your AI tool (Grok, Cursor, or Claude).
  2. 2Open a pack tagged Connector and check which apps it needs.
  3. 3Copy prompt and paste into your AI tool.
